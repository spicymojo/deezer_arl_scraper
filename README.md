# Deezer ARL Scraper and Web Server

This project consists of two Python scripts:

1. `deezer_arl.py`: This script scrapes a webpage for Deezer ARLs and generates an HTML page with the scraped information.

2. `deezer_web_server.py`: This script runs a web server that serves the HTML page generated by `deezer_arl.py`.

## deezer_arl.py

This script uses the `requests` and `beautifulsoup4` libraries to scrape a webpage for Deezer ARLs. The scraped information is then formatted into an HTML page.

The script defines a function `generate_html(url)`, which sends a GET request to the webpage, parses the response using BeautifulSoup, and finds all elements with the class 'ntable'. It then formats the information from these elements into an HTML page and writes it to `index.html`.

## deezer_web_server.py

This script runs a web server that serves the HTML page generated by `deezer_arl.py`. The server listens on port 8000 and serves the `index.html` file when the root URL (`/`) is requested.

## Dependencies

Both scripts require the `requests` and `beautifulsoup4` libraries. These dependencies can be installed using pip:

```bash
pip install -r requirements.txt
```
