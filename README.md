# Deezer ARL Scraper and Web Server

This project consists of two Python scripts:

1. `deezer_arl.py`: This script scrapes a webpage for Deezer ARLs and generates a Telegram message with the scraped information.

2. `deezer_arl_telegram_bot.py`: This script runs a Telegram bot that sends the ARL obtained by `deezer_arl.py`.

## Dependencies

Both scripts require some dependencies, define on the requirements file. These dependencies can be installed using pip:

```bash
pip install -r requirements.txt
```
